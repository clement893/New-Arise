## Method 1: From Your Project Folder (Easiest!)

1. **Open File Explorer** (press `Windows + E` or click the folder icon)
2. **Navigate to:** `C:\Users\cleme\New-Arise`
3. **Click in the address bar** at the top and type: `powershell` then press **Enter**

OR

3. **Right-click** in empty space inside the folder (not on a file)
4. **Click "Open in Terminal"** or **"Open PowerShell window here"**

---

## Method 2: From Start Menu

1. **Press the Windows key** (or click Start button)
2. **Type:** `powershell`
3. **Click "Windows PowerShell"** or **"PowerShell"**
4. **Navigate to your project:**
   ```powershell
   cd C:\Users\cleme\New-Arise
   ```

---

## Method 3: Run Dialog

1. **Press `Windows + R`**
2. **Type:** `powershell`
3. **Press Enter**
4. **Navigate to your project:**
   ```powershell
   cd C:\Users\cleme\New-Arise
   ```

---

## Method 4: Right-Click Start Button

1. **Right-click the Start button** (Windows logo)
2. **Click "Windows PowerShell"** or **"Terminal"**
3. **Navigate to your project:**
   ```powershell
   cd C:\Users\cleme\New-Arise
   ```
